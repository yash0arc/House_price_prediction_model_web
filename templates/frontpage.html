About the Project

This project predicts house prices based on key features like location, square footage, number of bedrooms and bathrooms, and area type. It uses a Ridge Regression model trained on cleaned and preprocessed housing data.

📂

Project Structure

Bengaluru_House_Data.csv – Dataset used.

main.ipynb / main.py – Scripts for preprocessing, training, evaluation.

model.pkl – Saved Ridge Regression model.

README.md – Project documentation.

🧩

Features Used

Location (One-hot encoded)

Area Type (One-hot encoded)

Size (Bedrooms)

Total Square Footage

Bathrooms

🧹

Data Preprocessing

Handled missing values and converted “2 BHK” to numeric.

Cleaned total_sqft values and standardized data.

Grouped sparse locations under “other”.

Removed outliers using bath ratio, sqft/room, IQR filtering.

🤖

Models Used

Linear Regression

Ridge Regression

Lasso Regression

Ridge Regression (α = 1) selected for best performance.

📊

Outlier Handling

Removed unrealistic data using statistical techniques:

• Bathroom-to-bedroom ratio

• Square feet per room

• IQR-based filtering for size, bath, and price

🧠

Model Selection

After cross-validation and hyperparameter tuning, Ridge Regression was finalized.

Performance optimized with alpha = 1.
